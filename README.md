# OpenClaw Skills

Collection of OpenClaw skills for autonomous AI assistant operations.

## Skills

| Skill | Description |
|-------|-------------|
| [auto-session-manager](auto-session-manager/) | Auto-manage OpenClaw session lifecycle |
| [claude-code-openclaw](claude-code-openclaw/) | Claude Code CLI wrapper for OpenClaw |
| [data-organizer](data-organizer/) | File organization and content extraction |
| [github-find-context7](github-find-context7/) | GitHub search and documentation lookup |
| [memory-consolidator](memory-consolidator/) | Memory file merging and summarization |
| [rag_to_qdrant](rag_to_qdrant/) | Directory monitoring to vector database |
| [reddit-ai-feeds-skill](reddit-ai-feeds-skill/) | Reddit AI news feeds |
| [web-content-learner](web-content-learner/) | Web content extraction and video transcription |

## Documentation Structure

Each skill contains:
- **README.md**: Bilingual documentation (English + Chinese)
- **SKILL.md**: Full English OpenClaw standard documentation
- **scripts/**: Python/bash scripts
- **references/**: Additional documentation

## Installation

Skills are automatically installed in `~/.openclaw/skills/` by OpenClaw.
